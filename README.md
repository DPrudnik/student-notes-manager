# Students Notes Manager

## Ідея проєкту

**Students Notes Manager** — це REST API застосунок для зберігання, організації та керування навчальними нотатками студентів. Система дозволяє створювати нотатки, привʼязувати їх до навчальних предметів, додавати теги та отримувати структуровану інформацію для ефективного навчання.

Проєкт розробляється в межах вивчення дисципліни **Node.js** та охоплює повний цикл розробки серверного програмного забезпечення: від ідеї та архітектури до тестування, CI/CD і оптимізації продуктивності.

---

## Основні можливості

- Створення, редагування та видалення нотаток
- Організація нотаток за навчальними предметами
- Додавання тегів до нотаток для зручного пошуку
- Отримання списку нотаток з фільтрацією
- REST API для взаємодії з клієнтськими застосунками

---

## Технологічний стек

- **Node.js**

- **TypeScript**

---

## Архітектура проєкту

Проєкт побудований за багатошаровою (layered) архітектурою:

- **Routes** — опис HTTP маршрутів
- **Controllers** — обробка HTTP-запитів і відповідей
- **Services** — бізнес-логіка застосунку
- **Models** — опис структур даних
- **Tests** — модульні, інтеграційні та E2E тести

Такий підхід забезпечує масштабованість, тестованість та зручність супроводу коду.

---

## Мета проєкту

Метою проєкту є:

- закріплення практичних навичок роботи з Node.js;
- вивчення принципів побудови серверних застосунків;
- ознайомлення з процесами тестування, CI/CD та оптимізації продуктивності;
- створення повноцінного навчального проєкту, який можна розширювати в майбутньому.

---

## Статус проєкту

Проєкт знаходиться в активній розробці та виконується поетапно відповідно до лабораторних робіт дисципліни Node.js.
